## Features
- **App**
  - Support fully .mp3,.wav files
  - Partly supports .ogg,.flac,.au files
  - Multiple User Accounts
  - Configurable via multiple settings
  - Build in web browser
  - Advanced Tag Editor
  - File Organizer
  - File Explorer
  - Multiple Libraries/Playlist support
  - Audio effects
  - System monitor ( CPU , RAM )
